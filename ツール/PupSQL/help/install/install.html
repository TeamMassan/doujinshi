インストール

PupSQLiteを使用するにはお手持ちのパソコンに.NET FrameWork 4とVisual C++ 2010再頒布パッケージがインストールされている必要があります。
必要ソフトがインストールされていれば pupsqlite_ver_○○.zip を解凍後、適当なフォルダ(My DocumentsやProgram Files)に保存してください。

必要に応じてデスクトップやスタート→プログラムなどにショートカットを作成してもよいでしょう。

バージョンアップ

PupSQLiteのバージョンアップは最新版のPupSQLiteを入手しインストールと同じ要領で古いPupSQLiteに上書きしてください。

最新版の入手は作者のホームページかVectorより可能です。

またPupSQLiteに同梱されているSystem.Data.SQLite.dllにはsqlite3.dllをC#で書き直したものでsqlite3.dll ver3.7.14と同等となります。
詳しくはこちらをご覧ください。

  • 作者のHP　https://www.eonet.ne.jp/~pup/

アンインストール

PupSQLiteのアンインストールはPupSQLite.exeが含まれるフォルダを削除するだけでOKです。
